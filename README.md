# Playwright Automation

This repository contains Playwright learning materials and automation examples, including a dedicated Playwright basics project under the 20_PlayWright_Basics folder.

## Repository contents

- 20_PlayWright_Basics/ - Playwright project with configuration, tests, and browser automation examples.
- 01_Basics/ to 08_2D_Arrays/ - JavaScript practice files used as a foundation for automation learning.
- package.json - Root Node.js project file with Playwright dependency.

## Getting started

1. Install dependencies:

```bash
npm install
```

2. Open the Playwright basics project:

```bash
cd 20_PlayWright_Basics
```

3. Verify Playwright is installed:

```bash
npx playwright --version
```

4. Run tests:

```bash
npx playwright test
```

## Notes

- The Playwright setup includes a configuration file and test folder for browser automation practice.
- The project uses the Playwright test runner for end-to-end automation examples.
- Node modules are ignored by Git to keep the repository lightweight.
